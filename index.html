कंप्यूटर शॉर्टकट की संपूर्ण गाइड

इस वेबसाइट पर आप MS Office और ऑपरेटिंग सिस्टम के सभी महत्वपूर्ण कीबोर्ड शॉर्टकट सीख सकते हैं। अपनी उत्पादकता को बढ़ाएं!

Microsoft Word शॉर्टकट

Ctrl + N: नया दस्तावेज़ बनाएं

Ctrl + O: मौजूदा दस्तावेज़ खोलें

Ctrl + S: दस्तावेज़ सहेजें

Ctrl + P: प्रिंट डायलॉग खोलें

Ctrl + B: टेक्स्ट को बोल्ड करें

Ctrl + I: टेक्स्ट को इटैलिक करें

Ctrl + U: टेक्स्ट को अंडरलाइन करें

Ctrl + Z: अंतिम क्रिया पूर्ववत करें

Ctrl + Y: पूर्ववत क्रिया को फिर से करें

F7: वर्तनी जांच

Microsoft Excel शॉर्टकट

Ctrl + Arrow Keys: डेटा के अंत तक जाएं

Ctrl + Shift + L: फिल्टर टॉगल करें

Alt + =: ऑटोसम फंक्शन डालें
